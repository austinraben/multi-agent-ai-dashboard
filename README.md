# multi-agent-ai-dashboard

An experimental full-stack application exploring **multi-agent AI** with CrewAI,
built while preparing for a software internship. A prototype, not a finished
product: a hands-on exploration of wiring an agent framework into a real
web stack.

## What it does
An AI-powered trivia game: it generates a random question through a
generative-AI API call, then uses AI agents to attempt the answer. Built to
experiment with agent orchestration, but answer accuracy was a known limitation
and the main open problem.

## Stack
- **Backend:** NestJS + MongoDB Atlas
- **AI:** Python + CrewAI (multi-agent orchestration)
- **Frontend:** React

## Status
Prototype / learning project.
